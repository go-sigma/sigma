# What is sigma?

Sigma is a painless self-hosted all in one software development service, it includes OCI artifact manager, garbage collection, namespace quota, multiarch artifact, OCI image build. It is similar to [Harbor](https://goharbor.io/), but [distribution](https://github.com/distribution/distribution) is implement by itself, all of the service can be startup with one command.
